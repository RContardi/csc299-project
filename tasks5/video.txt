This video demonstrates the tasks5 project, which showcases GitHub's spec-kit methodology applied to building a task management system.

## Key Demonstration Points

✅ Complete specification suite before implementation
✅ Modular architecture with clear separation of concerns
✅ Test-driven development with >80% coverage
✅ Zero external dependencies (standard library only)
✅ Comprehensive documentation and traceability
✅ Constitutional principles enforced throughout

## Commands Demonstrated

```powershell
# Installation
cd TaskManager
python -m pip install -e .

# Basic usage
task-manager add "Buy groceries" --description "Milk, eggs, bread"
task-manager list
task-manager search "groceries"
task-manager complete 1

# Natural language input
python say_app.py
.\say.ps1 put buy milk on my tasks

# Running tests
python -m unittest discover -v

# Viewing specifications
cat specs\001-task-manager\spec.md
cat specs\001-task-manager\plan.md
```

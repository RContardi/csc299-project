# Stride Installer - Quick Start Guide

## What is Stride?
Stride is an AI-powered task manager that lets you manage your to-do list through natural conversation. Just chat with it like you would with a personal assistant!

## Installation (Simple 3-Step Process)

### Step 1: Extract the ZIP
- Right-click `Stride-Installer.zip`
- Select "Extract All..."
- Choose a location (like `C:\Program Files\Stride` or `Documents\Stride`)

## What You Need

**Python 3.8 or higher**
- Download from: https://www.python.org/downloads/
- During install, CHECK "Add Python to PATH"
- Restart your computer after installing

**OpenAI API Key** (Free to create)
1. Go to: https://platform.openai.com/
2. Sign up for a free account
3. Go to API Keys section
4. Click "Create new secret key"
5. Copy the key (starts with `sk-`)
6. Paste it when the installer asks

## After Installation

1. Double-click the **Stride** icon on your desktop
2. The app will open with an AI chat interface
3. Start managing your tasks with natural language!

## Example Commands
- "Add a task to finish my project report"
- "Complete task 1"
- "Show me all my tasks"
- "Remove all completed tasks"
- "What did I delete yesterday?"

## Need Help?

- **Can't find Python?** The installer will give you the download link
- **API Key not working?** Make sure you copied the entire key
- **Shortcut won't open?** Try running `main.py` in the folder directly

See `INSTALLATION_GUIDE.md` in the ZIP for detailed troubleshooting.
